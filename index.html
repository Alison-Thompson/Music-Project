Description

This is a web project that, using JavaScript, can create musical chord progressions of varrying lengths. This can be used as a song writing tool or just for fun. What ever you deciede to do with the code make sure you try generating many progressions and listen to them to find what you like. When you've found a progression you like make sure to save it so you don't lose it. Happy generating.
